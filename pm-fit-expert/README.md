# FitLine Health AI (PM FitExpert)

전문적인 인바디 데이터 분석과 독일 PM 프리미엄 영양 솔루션을 결합한 하이엔드 헬스케어 리포트 서비스입니다.

## 🏛️ 주요 기능
- **AI 인바디 판독**: 인바디 결과지 사진을 촬영하면 AI가 핵심 수치(체중, 근육, 지방 등)를 자동으로 정밀 추출합니다.
- **9종 정밀 체형 진단**: BMI, 골격근량, 체지방률 데이터를 기반으로 과학적인 9가지 체형 유형을 진단합니다.
- **임상 기반 맞춤 처방**: 진단된 체형에 최적화된 독일 PM 제품 조합과 생활 습관 가이드를 제공합니다.
- **럭셔리 클리니컬 디자인**: 하이엔드 의료기관 수준의 세련된 UI/UX와 시각적 리포트를 제공합니다.

## 🚀 배포 방법 (GitHub Pages)
1. GitHub 저장소 생성 (`pm-fit-expert`)
2. 로컬 저장소 연결:
   ```bash
   git remote add origin https://github.com/사용자아이디/pm-fit-expert.git
   ```
3. 라이브 배포 실행:
   ```bash
   npm run deploy
   ```

## 📱 기술 스택
- **Core**: Vanilla JavaScript (ES6+), HTML5, CSS3
- **Build Tool**: Vite
- **AI Library**: Tesseract.js (OCR Engine)
- **Deployment**: GitHub Pages
